标题

1-6个#或 ===或 -

字体 *或_

1个代表斜体
2个粗体
3个粗斜体

线

分隔线

一行中用三个以上的星号、减号、底线来建立一个分隔线，行内不能有其他东西。你也可以在星号或是减号中间插入空格

删除线

如果段落上的文字要添加删除线，只需要在文字的两端加上两个波浪线 ~~ 即可
哈哈哈哈哈

脚注

创建脚注格式类似这样 [^ AAA]。

[^AAA]: 脚注

列表

无序列表

无序列表使用星号(*)、加号(+)或是减号(-)作为列表标记，这些标记后面要添加一个空格

  • 第一项
  • 第二项
  • 第三项

    有序列表

    有序列表使用数字并加上 . 号来表示，如：
  1. 第一项
  2. 第二项
  3. 第三项

    列表嵌套

列表嵌套只需在子列表中的选项前面添加四个空格即可：

  1. 第一项：
    • 第一项嵌套的第一个元素
    • 第一项嵌套的第二个元素
  2. 第二项：
    • 第二项嵌套的第一个元素
    • 第二项嵌套的第二个元素

      区块

      区块

      Markdown 区块引用是在段落开头使用 > 符号 ，然后后面紧跟一个空格符号：

      区块
      qukuai

      嵌套区块

      第一个

      第二个

代码

如果是段落上的一个函数或片段的代码可以用反引号把它包起来（在1旁边那个键），例如：
printf() 函数

//三个反引号来包含代码区块

tab键或四个空格

//四个空格

链接

[名字] (地址)
例子
我的博客
<地址 >
https://www.cnblogs.com/sellun/

图片

! [alt 属性文本 ] (https://pic.cnblogs.com/avatar/2426787/20210619101310.png)
! [alt 属性文本 ] (https://pic.cnblogs.com/avatar/2426787/20210619101310.png “可选标题”)
< img src=”https://pic.cnblogs.com/avatar/2426787/20210619101310.png" width=”50%”>

表格

Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。

表头 表头
单元格 单元格
单元格 单元格

我们可以设置表格的对齐方式：

-: 设置内容和标题栏居右对齐。
:- 设置内容和标题栏居左对齐。
:-: 设置内容和标题栏居中对齐。
实例如下：

| 左对齐 | 右对齐 | 居中对齐 |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
左对齐 右对齐 居中对齐
单元格 单元格 单元格
单元格 单元格 单元格

转义

Markdown 支持以下这些符号前面加上反斜杠来帮助插入普通的符号：

\   反斜线
`   反引号
*   星号
_   下划线
{}  花括号
[]  方括号
()  小括号
#   井字号
+   加号
-   减号
.   英文句点
!   感叹号
